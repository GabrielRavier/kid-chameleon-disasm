# This list is highly incomplete, and just included to
# put known RAM addresses into one place.

FFF81C	X camera position (long)
FFF820	Y camera position (long)
FFF824	X camera background position? (long)
FFF828	Y camera background position? (long)
FFFAD1	game paused menu option (byte)
FFFBC4	Pointer to current Keypress in Demo
FFFBC9	Demo Mode flag (for input)
FFFBCA	Game Mode:
	00 - SegaScreen
	04 - IntroSequence1
	08 - TitleCard
	0C - InGame        ; also Results screen
	10 - DemoPlay
	14 - OptionMenu
	18 - IntroSequence2
	1C - IntroSequence3
	20 - IntroSequence4
	24 - IntroSequence5
	28 - IntroSequence6 ; is also TitleScreen if intro played completely
	2C - TitleScreen
	30 - EndSequence
FFFBD4	address of map header
FFFC38	two player flag ?
FFFC3E 	number of lives
FFFC40	Number of hit-points filled
FFFC42 	Number of Diamonds
FFFC44	Current LevelID
FFFC46 	Costume/helmet
	00 - None
	01 - Skycutter
	02 - Cyclone
	03 - Red Stealth
	04 - Eyeclops
	05 - Juggernaut
	06 - Iron Knight
	07 - Berzerker
	08 - Maniaxe
	09 - Micromax
FFFC48	Number of continues
!FFFDC7	HACK: used as level select indicator (byte)
FFFDC8	selected sub-option 1 in options menu (byte)
FFFDC9	selected sub-option 2 in options menu (byte)
FFFDCA	selected sub-option 3 in options menu
FFFDCC	selected option in options menu

FF0DB8	Menu timer (word)
FF0DBD	main menu option
FF0DC1	main menu option * 2
FF2596	main menu option selector Y position

# Bitmask for controller keys: SACBRLDU
FFF80C	Ctrl_1
FFF80D	Ctrl_1_Held (currently held down)
FFF80E	Ctrl_1_Pressed (newly pressed this frame)
FFF80F	Ctrl_2
FFF810	Ctrl_2_Held
FFF811	Ctrl_2_Pressed
# depending on input mode (i.e. currently active player?):
FFF812	Ctrl_Held
FFF813	Ctrl_Pressed

FF0DA0	interesting block behavior (?)
FF2448	Object layout
FF4E58	Some Palette Buffer	(0x80 bytes)
FF503A	Mappings for blocks
FF52C2	Level tile layout
FF73C8-FF77AF	deformation array
FF7D8E	Some Palette Buffer	(0x40 bytes)
FFA652	Level layout buffer
